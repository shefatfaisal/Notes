subnet has 3 things

- network address/id
	- address to identify the network
	- called routing prefix as well
- host range
	- the collection of ip addresses give to nodes on the network
	- each node must have a unique address in the range, no duplicates
- broadcast address/id
	- address to transmit a packet to all devices on the ip subnet
	- knows as a directed broadcast

**Example subnet addressing

**Network ID       Host Range    Broadcast id**
192.168.1.0         192.168.1.1 -      192.168.1.255
			   192.168.1.254
	**Always in this order**
