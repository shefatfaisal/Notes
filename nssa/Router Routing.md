Every router interface

- has a name(example:f0/1)
- has a MAC address
- has and IP address(provided by me default gateway)
- is in a different network from the other interfaces on the router


**Types of routes**
- Connected (c)
	- installed when giving a router interface its ip address
	- indicates that a router interface is connected to that network
* installed with a static route command
* Manually tell the router how to route the destination network
