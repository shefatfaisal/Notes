## 1) Virtual Domain and Network Infrastructure

Projects where you build a small enterprise style network in a lab.

- Active Directory domain lab: domain controller, users and groups, OUs, GPOs, login scripts, password policies
    
- DHCP and DNS services: scopes, reservations, forwarders, dynamic DNS updates, troubleshooting name resolution
    
- pfSense firewall gateway: NAT, VLANs, firewall rules, port forwarding, VPN (OpenVPN or WireGuard)
    
- Multi subnet lab: Windows client plus Linux client, routing between VLANs, access control between networks
    
- “Enterprise” add ons: WSUS patching, file shares with NTFS permissions, RADIUS for Wi Fi auth, basic monitoring
    
- Git server setup on Linux: SSH keys, permissions, repo structure, backups, hooks, basic CI style checks
